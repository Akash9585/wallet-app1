# Wallet App - Full Stack React + Express.js

Complete banking wallet application with real-time OTP authentication and dynamic data visualization.

## Features

- Real Time OTP Authentication (Express Backend)
- Pie Chart Visualization (Recharts)
- Mobile Responsive Design
- Data Input Screen (Salary + Expenses)
- Transaction Management Modal
- Full Backend API Integration

## Tech Stack

**Frontend:**
- React.js
- Vite (Build Tool)
- Tailwind CSS
- Recharts (Charts)
- Lucide React (Icons)
- Date-fns (Date Handling)

**Backend:**
- Express.js
- CORS Enabled
- JSON API Endpoints
